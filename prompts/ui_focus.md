# UI Replication Focus

Pay special attention to:

- **UI Elements**: Buttons, forms, inputs, dropdowns, modals, menus
- **Layout**: Positioning, spacing, alignment, grid/flex structure
- **Colors**: Hex codes if possible, or descriptive (e.g., "primary blue #3B82F6")
- **Typography**: Font families, sizes, weights, text content
- **Interactions**: Clicks, hovers, transitions, animations shown
- **States**: Default, hover, active, disabled, error states

**Audio Priority: MEDIUM** - Narrator often guides interactions.

Example correlation:
- Audio at 3.1s: "Click the blue login button"
- Visual at frame 3: Shows cursor hovering over blue button labeled "Login"
- Correlation: "Blue button (#2563EB) with 'Login' text, rounded corners, at bottom-right"
